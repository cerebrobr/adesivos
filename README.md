# Dev Stickers

Uma coleção de adesivos para desenvolvedores.

## Estrutura

* `print`: arquivos para impressão.
* `view`: arquivos para visualização.

> Nota: no momento da impressão verificar a resolução (300dpi) e escala de cores (CMYK).
